# Predicting-US-Flight-Demand
A graduate-level university project which combines supervised regression and time-series forecasting with clustering to produce a two-stage analytical picture of US passenger demand. View the following [report](https://dan-a-rama.neocities.org/project-pdfs/Where%20to%20Fly%20Next%20-%20Predicting%20and%20Segmenting%20US%20Domestic%20Passenger%20Demand%20%5BReport%5D.pdf) and [presentation PDF](https://dan-a-rama.neocities.org/project-pdfs/Where%20to%20Fly%20Next%20-%20Predicting%20and%20Segmenting%20US%20Domestic%20Passenger%20Demand%20%5BPresentation%5D.pdf) for additional information!

The dataset can be found [HERE](https://www.kaggle.com/datasets/aliafzal9323/us-domestic-airline-airfare-dataset-20082025) and has also been included in this repository.

# Instructions To Run The Project
The following steps provide the easiest way to run the project:

1. Download the project files and run the .ipynb file in Visual Studio Code
2. Ensure that Python is installed on your system.
3. Select `Run All`, select a Python installation to run all cells. You will be prompted to install extensions and kernels. Accept all of them.
4. Wait for all cells to complete and view the results!
